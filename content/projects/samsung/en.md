# Coupon Redemption and User Synchronization Platform

## Overview

Project of a **promotional campaign management platform**, enabling **coupon redemption linked to CPF** and **synchronization of users** who purchased tickets for a major show in Brazil, focusing on **high conversion, data integrity, and security**.

The solution was designed to capture leads, link users and coupons, and integrate data between the **React frontend** and **VTEX MasterData**, ensuring consistency and reliability during the Samsung S24 launch.

**Strategic keywords:** promotional campaign, coupon redemption, user synchronization, React frontend, Node.js BFF, VTEX MasterData, secure checkout, high conversion, user experience, data integrity.

---

## Architecture

The architecture was designed to **efficiently integrate frontend and backend**, focusing on **lead capture, checkout security, and real-time data consumption**.

### Architectural Pattern

- Backend-for-Frontend (BFF)  
- Decoupled front-back integration  
- Synchronous data processing for lead capture  
- Data protection and consistency in coupon application  

### Technologies and Services Used

- **Node.js (BFF)** for API routing and coupon linking logic.  
- **React** for building the user interface and lead capture forms.  
- **VTEX MasterData** for registration, synchronization, and tracking of coupon usage.  
- **Secure Checkout** with additional protection layers and validation.  
- Monitoring and logging for campaign auditing and error resolution.
